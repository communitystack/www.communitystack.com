#DocsThursday

Create and contribute to open source documentation on the 3rd Thursday of every month.

How to Participate

  1. Choose a Location Join from home, work, a participating meetup, or even a local coffee shop. You just need a laptop and Internet.
  2. Pick a Project Choose an open source docs site from this website or just find the repo for your favorite project's docs.
  3. Submit Issues & PRs Open GitHub Issues for missing or unclear information or open PRs to fix typos, add missing features, or elaborate sections.
